# 文字

```
*文字*斜体，
**文字**加粗，
“*”可用“_”代替，也能实现斜体和加粗。

删除划线内容：~~要划除的行内内容~~

文字间换行：在文字前输入

文字下有分割线：---或者***，使用时候注意要在文字下和---之间有空行。

文字底纹：`文字内容`
```
